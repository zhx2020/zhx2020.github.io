Linux常用命令

ZHX2020
2020-05-28 (Updated: 2020-08-17)
Java面试
linux

基础指令

关机命令

ls：列出文件和文件夹

pwd：打印当前工作目录

cd：切换目录

mkdir：创建目录

touch：创建文件

cp：复制

mv：移动、剪切

rm：删除

vim：文本编辑器

>/>>：输出重定向

cat：查看文件内容

进阶指令

df：查看磁盘空间

free：查看内存使用情况

head：查看一个文件的前n行，不指定n，默认查看前10行

tail：查看一个文件的末n行，不指定n，默认查看后10行

less：查看文件，以较少的内容进行输出

wc：统计文件内容信息

date：操作时间和日期

cal：操作日历

clear / ctrl + l：清除信息

|：管道

高级指令

hostname：操作服务器的主机名

id：查看一个用户的一些基本信息

whoami：显示当前登录的用户名

ps：查看服务器的进程信息

top：查看服务器的进程占的资源

du：查看目录的真实大小

find：查找文件

systemctl：控制一些软件的服务（启动、停止、重启）

kill：杀死进程

ifconfig：操作网卡相关

reboot：重新启动计算机

shutdown：关机

uptime：输出计算机的持续在线时间

uname：获取计算机操作系统相关信息

netstat：查看网络连接状态

man：手册

用户管理

三个文件

useradd：添加用户

usermod：修改用户

passwd：设置密码

userdel：删除用户

用户组管理

groupadd：添加用户组

groupmod：修改用户组

groupdel

权限管理

权限介绍

身份介绍

linux权限介绍

chmod：权限设置

属主和属组设置

chown：更改文档的所属用户

chgrp：更改文档的所属组（主组）

更改所属用户和所属组

其它命令

ping：检测当前主机和目标主机之间的连通性

su：切换用户

tar：解压

&：用在一个命令的最后，可以把这个命令放到后台执行

ctrl + z：可以将一个正在前台执行的命令放到后台，并且暂停

jobs：查看当前有多少在后台运行的命令

fg：将后台中的命令调至前台继续运行

bg：将一个在后台暂停的命令，变成继续执行
